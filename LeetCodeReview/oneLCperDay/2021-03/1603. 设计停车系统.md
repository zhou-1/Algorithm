## 1603. 设计停车系统 【简单】      
[设计]     
https://leetcode-cn.com/problems/design-parking-system/     

### 数组   
```
class ParkingSystem {
    //创建一个数组
    int[] sys;

    public ParkingSystem(int big, int medium, int small) {
        this.sys = new int[]{big, medium, small};
    }
    
    public boolean addCar(int carType) {
        if(carType == 1) {
            this.sys[0]--;
            if(this.sys[0] < 0) {
                return false;
            } else {
                return true;
            }
        } else if(carType == 2) {
            this.sys[1]--;
            if(this.sys[1] < 0) {
                return false;
            } else {
                return true;
            }
        } else {
            this.sys[2]--;
            if(this.sys[2] < 0) {
                return false;
            } else {
                return true;
            }
        }
       
    }
}

```
