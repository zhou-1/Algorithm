## 74. 搜索二维矩阵 【中等】     
[数组，二分查找]     
https://leetcode-cn.com/problems/search-a-2d-matrix/    

### 最直观：全部遍历     
```
public boolean searchMatrix(int[][] matrix, int target) {
        int m = matrix.length, n = matrix[0].length;

        if(matrix == null || m == 0 || n == 0) {
            return false;
        }

        for(int i = 0; i < m; i++) {
            for(int j = 0; j < n; j++) {
                if(matrix[i][j] == target) {
                    return true;
                }
            }
        }

        return false;
}
```
