## Solution     

### post order traversal      
Great solution of the post order traversal in (right, left, root) order!

code:
```
private TreeNode prev = null;

public void flatten(TreeNode root) {
    if (root == null)
        return;
    flatten(root.right);
    flatten(root.left);
    root.right = prev;
    root.left = null;
    prev = root;
}
```
